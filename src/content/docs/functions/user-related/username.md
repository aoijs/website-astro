`$username` will return a user's username.

## Usage

```php
$username[userID?]
```

## Parameters

| Field   | Type    | Description  | Required |
| ------- | ------- | ------------ | :------: |
| userID? | integer | The user ID. |  false   |

## Example(s)

This will return your username:

```javascript
bot.command({
  name: "username",
  code: `
  $username[$authorID]
  `,
});
```
