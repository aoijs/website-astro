`$userID` will return a given user's ID.

## Usage

```php
$userID[user]
```

## Parameters

| Field | Type           | Description                                                      | Required |
| ----- | -------------- | ---------------------------------------------------------------- | :------: |
| user  | string, number | The name of the user who you want the user ID to be returned of. |   true   |

## Example(s)

This will return your user ID:

```javascript
bot.command({
    name: 'userID',
    code: `
  $userID[$username]
  `
});
```
