`$deleteCommand` will delete the initial command message.

## Usage

```php
$deleteCommand
```

## Example(s)

This will delete the initial command message:

```javascript
bot.command({
  name: "deleteCommand",
  code: `
  Hello!
  $deleteCommand
  `,
});
```
