`$spawnShard` will spawn a shard, requires sharding.

## Usage

```php
$spawnShard[shard]
```

## Parameters

| Field | Type   | Description       | Required |
| ----- | ------ | ----------------- | :------: |
| shard | number | Amount of shards. |   true   |

## Example(s)

This will spawn shard `1`:

```javascript
bot.command({
    name: 'spawnShard',
    code: `
   $spawnShard[1]`
});
```
