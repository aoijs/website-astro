`$deleteArray` will delete a previously created array.

## Usage

```php
$deleteArray[name]
```

## Parameters

| Field | Type   | Description | Required |
| ----- | ------ | ----------- | :------: |
| name  | string | Array name. |   true   |

## Example(s)

This will delete the array "array":

```javascript
bot.command({
  name: "deleteArray",
  code: `
  $deleteArray[array]
  $createArray[array;Hello;This;Is;Great]
  `,
});
```
