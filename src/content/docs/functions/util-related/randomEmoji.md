`$randomEmoji` will return a random emoji ID.

## Usage

```php
$randomEmoji
```

## Example(s)

This will return a random emoji ID of your guild:

```javascript
bot.command({
    name: 'randomEmoji',
    code: `
  $randomEmoji
  `
});
```
