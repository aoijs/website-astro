`$createCache` will create custom cache.

## Usage

```php
$createCache[type;name;options]
```

## Parameters

| Field   | Type   | Description    | Required |
| ------- | ------ | -------------- | :------: |
| type    | string | Cache type.    |   true   |
| name    | string | Cache name.    |   true   |
| options | string | Cache options. |   true   |
