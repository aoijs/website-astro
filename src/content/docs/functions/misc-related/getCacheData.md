`$getCacheData` will return cache data.

## Usage

```php
$getCacheData[type;name;key;option?]
```

## Parameters

| Field   | Type   | Description | Required |
| ------- | ------ | ----------- | :------: |
| type    | string |             |   true   |
| name    | string |             |   true   |
| key     | string |             |   true   |
| option? | string |             |  false   |
