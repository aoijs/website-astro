`$modulo` operation / the remainder when dividing.

## Usage

```php
$modulo[num;num]
```

## Parameters

| Field | Type   | Description                         | Required |
| ----- | ------ | ----------------------------------- | :------: |
| num   | number | Numbers to return the remainder of. |   true   |

## Example(s)

This will return `2` as it's the remainder of `5 % 3`:

```javascript
bot.command({
  name: "modulo",
  code: `
  $modulo[5;3]
  `,
});
```
