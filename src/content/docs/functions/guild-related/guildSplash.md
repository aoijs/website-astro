`$guildSplash` will return a guild's invite background (if unlocked).

## Usage

```php
$guildSplash[guildID?]
```

## Parameters

| Field    | Type    | Description          | Required |
| -------- | ------- | -------------------- | :------: |
| guildID? | integer | The ID of the guild. |  false   |

## Example(s)

This will return the guild's invite background (if unlocked):

```javascript
bot.command({
  name: "guildSplash",
  code: `
  $guildSplash[$guildID]
  `,
});
```
