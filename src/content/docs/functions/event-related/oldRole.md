`$oldRole` holds data of the role before it was updated, used in role update event, or else any data will be empty.

## Usage

```php
$oldRole[option]
```

## Parameters

| Field  | Type   | Description         | Required |
| ------ | ------ | ------------------- | :------: |
| option | string | Option to retrieve. |   true   |
