`$newVariable` gets the data from variable events.

## Usage

```php
$newVariable[option;sep?]
```

## Parameters

| Field  | Type   | Description         | Required |
| ------ | ------ | ------------------- | :------: |
| option | string | Option to retrieve. |   true   |
| sep    | string | Separator.          |  false   |
