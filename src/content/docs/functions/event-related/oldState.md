`$oldState` holds data for the user voice state before it was updated (voiceStateUpdate event).

## Usage

```php
$oldState[option]
```

## Parameters

| Field  | Type   | Description         | Required |
| ------ | ------ | ------------------- | :------: |
| option | string | Option to retrieve. |   true   |
