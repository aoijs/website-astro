`$oldMessage` holds the content of the message before it was updated. (if any)

## Usage

```php
$oldMessage[option]
```

## Parameters

| Field  | Type   | Description         | Required |
| ------ | ------ | ------------------- | :------: |
| option | string | Option to retrieve. |   true   |
