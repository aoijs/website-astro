`$shuffleQueue` will shuffle the current queue. 

## Usage

```php
$shuffleQueue
```

## Example(s)

This will shuffle the current queue:

```javascript
bot.command({
    name: 'shuffleQueue',
    code: `
    $shuffleQueue
  `
});
```
